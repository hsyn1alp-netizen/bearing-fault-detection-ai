# Bearing Fault Detection System

## Overview
This project detects bearing faults using multi-sensor data fusion.

- Vibration data → MLP (MATLAB)
- Image data → CNN (TensorFlow)
- Fusion → Weighted decision-level fusion
- Anomaly Detection → Z-score

## Features
- Real-time fault detection
- Multi-sensor fusion
- Embedded C model (MATLAB Codegen)
- Logging system

## Technologies
- MATLAB (MLP)
- Python (TensorFlow)
- C (Embedded AI)
- Sensor Fusion
- Anomaly Detection

## Project Structure
- vibration_model → C-based neural network
- image_model → CNN (Python)
- fusion → decision system
